get_folder_content.xml

This SOAP request allows you to query the content of any folder in the xincoDMS repository by submitting the folder's internal ID.

The response contains the XincoCoreNode object with its designation, parent ID and the directly subordinated data and folders. Therefore this request is the basis of navigation in the repository. It provides you with the IDs of files and subfolders required by many other operations.

Basic parameters

The text content of the following element nodes has to be adjusted:

XPath queryValue
/soapenv:Envelope/soapenv:Body/ns1:getXincoCoreNode/in0/id Internal ID of the desired folder (e.g. 1001)

Predefined values in the repository:
1 = Root (xincoRoot)
2 = Trash
3 = Temp
4 = News
/soapenv:Envelope/soapenv:Body/ns1:getXincoCoreNode/in1/username Username of the user performing the operation
/soapenv:Envelope/soapenv:Body/ns1:getXincoCoreNode/in1/userpassword Password of the user performing the operation

Hint: Each of the XPath queries above will return an element node. To modify its content you will have to go down to its text node which is typically the first child.
