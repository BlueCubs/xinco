get_file_info.xml

This SOAP request allows you to query detailed information on a data object by submitting its internal ID.

The response contains the XincoCoreData object with its designation, parent ID and meta-information like the XincoAddAttributes (e.g. Description, Keywords) and XincoCoreLog entries associated with this item.

The XincoCoreLog elements are especially useful in combination with the download_file.xml request where a certain log entry determines the version of the data object that should be downloaded. This enables you to download older versions of files with Revision Model enabled.

Basic parameters

The text content of the following element nodes has to be adjusted:

XPath queryValue
/soapenv:Envelope/soapenv:Body/ns1:getXincoCoreData/in0/id Internal ID of the desired data object (e.g. 1001)
/soapenv:Envelope/soapenv:Body/ns1:getXincoCoreData/in1/username Username of the user performing the operation
/soapenv:Envelope/soapenv:Body/ns1:getXincoCoreData/in1/userpassword Password of the user performing the operation

Hint: Each of the XPath queries above will return an element node. To modify its content you will have to go down to its text node which is typically the first child.
