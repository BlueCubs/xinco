remove_ace.xml

This SOAP request allows you to remove permissions on certain folders or data objects.

The response contains the boolean value "true" if the ACE has been removed successfully or "false" in a case of failure.

Basic parameters

The text content of the following element nodes has to be adjusted:

XPath queryValue
/soapenv:Envelope/soapenv:Body/ns1:removeXincoCoreACE/in0/id Internal ID of the ACE to be removed (e.g. 1001). This can be obtained by the get_file_info.xml or get_folder_content.xml request.
/soapenv:Envelope/soapenv:Body/ns1:removeXincoCoreACE/in1/username Username of the user performing the operation
/soapenv:Envelope/soapenv:Body/ns1:removeXincoCoreACE/in1/userpassword Password of the user performing the operation

Hint: Each of the XPath queries above will return an element node. To modify its content you will have to go down to its text node which is typically the first child.
