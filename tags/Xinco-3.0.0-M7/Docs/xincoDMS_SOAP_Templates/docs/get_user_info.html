get_user_info.xml

This SOAP request enables you to query detailed user information by just transmitting the username and password of the respective user.

The response contains the XincoUser object with first name, last name, e-mail address, status and group memberships.

Basic parameters

The text content of the following element nodes has to be adjusted:

XPath queryValue
/soapenv:Envelope/soapenv:Body/ns1:getCurrentXincoCoreUser/in0 Username of the desired user (e.g. admin)
/soapenv:Envelope/soapenv:Body/ns1:getCurrentXincoCoreUser/in1 Password for the specified user

Hint: Each of the XPath queries above will return an element node. To modify its content you will have to go down to its text node which is typically the first child.
